	Social Share allow visitors to share job link to thier social network site.

== Description ==

Social Share allows to display popular social sharing button on single job page, so visitors can share job link with their social network site with friends & others.

you can add your own social button by filter hook.


### Job Board Manager - Social Share by http://pickplugins.com
* [Live Demo &raquo;](http://www.pickplugins.com/demo/job-board-manager/)
* [Get Job Board Manager &raquo;](https://wordpress.org/plugins/job-board-manager/)




== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Job Board Manager</strong>" activate it.<br />
